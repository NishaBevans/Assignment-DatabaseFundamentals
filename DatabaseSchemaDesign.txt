Authors
------------------------
| AuthorID (PK)       |
| FirstName           |
| LastName            |
| Biography           |
| BirthDate           |
------------------------

Books
------------------------
| BookID (PK)         |
| Title               |
| AuthorID (FK)      |
| Genre               |
| PublishedYear       |
| ISBN                |
| Price               |
| StockQuantity       |
------------------------

Customers
------------------------
| CustomerID (PK)     |
| FirstName           |
| LastName            |
| Email (UNIQUE)      |
| PhoneNumber         |
| Address             |
| JoinDate            |
------------------------

Transactions
------------------------
| TransactionID (PK)  |
| CustomerID (FK)     |
| BookID (FK)         |
| TransactionDate      |
| Quantity            |
| TotalAmount         |
------------------------
